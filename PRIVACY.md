# Privacy

Release Downloader stores data locally on the user's device.

## Local data

- Last selected download directory
- Up to five recently searched repositories
- Optional GitHub PAT, only when OS secure storage is available

## Network use

- Repository and release metadata are requested from the GitHub API
- Selected asset downloads are requested from approved GitHub/GitHubusercontent HTTPS hosts

## Data sharing

- The app does not operate its own backend service
- The app does not intentionally transmit analytics or advertising identifiers

## User responsibility

If you enter a GitHub PAT, you are responsible for using a minimal-scope token and protecting your device account.
